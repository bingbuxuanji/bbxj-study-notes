## 🗺️ 学习路线

1. **电力电子技术是什么** — 定义、应用场景、四种基本变换
2. **电学概念回顾** — 电压、电流、功率、电感、电容
3. **半导体器件** — 从 PN 结到各种电力电子开关器件
4. **整流电路（AC→DC）** — 半波/桥式/三相，不可控/可控，电阻/电感负载
5. **逆变电路（DC→AC）** — 电压型/电流型、负载特性影响、频率控制
6. **斩波电路（DC→DC）** — 升压/降压
7. **交流变换（AC→AC）** — 变频、调压
8. **换流技术** — 外部换流/自换流分类体系、四种换流方式、强迫换流子类（电压换流/电流换流）
