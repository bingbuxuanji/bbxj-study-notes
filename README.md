## 🗺️ 学习路线

### 电力电子技术
1. **电力电子技术是什么** — 定义、应用场景、四种基本变换
2. **电学概念回顾** — 电压、电流、功率、电感、电容
3. **半导体器件** — 从 PN 结到各种电力电子开关器件
4. **整流电路（AC→DC）** — 半波/桥式/三相，不可控/可控，电阻/电感负载
5. **逆变电路（DC→AC）** — 电压型/电流型、负载特性影响、频率控制
6. **斩波电路（DC→DC）** — 升压/降压
7. **交流变换（AC→AC）** — 变频、调压
8. **换流技术** — 外部换流/自换流分类体系、四种换流方式

### 信号分析及处理

1. **信号分类与系统性质** — 连续/离散、线性/时不变/因果
2. **奇异信号** — $\delta(t)$、$u(t)$、$\delta'(t)$ 及其性质
3. **卷积** — 连续/离散卷积、卷积定理（时域卷积 = 频域乘积）
4. **傅里叶级数（FS）** — 周期信号 → 离散线谱
5. **傅里叶变换（FT）** — 非周期信号 → 连续谱，8 大必背变换对
6. **拉普拉斯变换（LT）** — FT 的推广，ROC 与稳定性判据
7. **Z 变换（ZT）** — 离散域的核心工具，ROC 与因果/稳定判据
8. **DFT / FFT** — 离散傅里叶变换、时频对偶关系
9. **数字滤波器** — FIR vs IIR、设计方法（脉冲响应不变/双线性/窗函数）
10. [📋 常用变换速查表](信号分析/附录_常用变换表.md) — 🔥 傅里叶/拉普拉斯/Z 变换公式大全

### 概率论与数理统计
1. **从赌博到科学** — 样本空间、事件、概率的朴素定义
2. **为什么要公理化** — 概率的三条公理与基本性质
3. **条件概率与贝叶斯** — 新信息如何改变判断
4. **随机变量** — 把"结果"变成"数字"
5. **常见分布** — 二项/泊松/正态...无处不在的曲线
6. **数字特征** — 期望、方差、协方差
7. **大数定律与中心极限定理** — 概率论的"两大支柱"
8. **统计推断** — 参数估计、置信区间、假设检验

## 📊 教程特色

- ✅ **零基础友好**：只需高中物理/数学基础
- ✅ **例题驱动**：每个概念由一个具体问题引出，先看到问题再引入工具
- ✅ **波形图/图表可视化**：所有关键电路和概率分布都配有 Python 生成的图，直观理解
- ✅ **循序渐进**：先讲"是什么"→ 再"为什么"→ 最后"怎么用"
- ✅ **公式推导零跳步**：关键公式每一步都写出来，不假设读者"显然知道"
- ✅ **跨学科概念完整引入**：从其他学科借用的概念（集合、积分），从零讲起
