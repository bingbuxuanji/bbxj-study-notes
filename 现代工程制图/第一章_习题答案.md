## 1-1 填空选择题

### 1-1-1 填空题

| 题号 | 答案 |
|:---:|------|
| (1) | 将 A0 幅面的图纸裁切三次，应得到 **8** 张图纸，其幅面代号为 **A3**。 |
| (2) | 要获得 A4 幅面的图纸，需将 A0 幅面的图纸裁切 **4** 次，可得到 **16** 张图纸。 |
| (3) | A4 幅面的尺寸（B×L）是 **210×297**；A3 幅面的尺寸（B×L）是 **297×420**。 |
| (4) | 用放大一倍的比例绘图，在标题栏的"比例"栏中应填写 **2∶1**。 |
| (5) | 1∶2 是放大比例还是缩小比例？ **缩小比例**。 |
| (6) | 若采用 1∶5 的比例绘制一个直径为φ40mm 的圆时，其绘图直径为 **φ8** mm。 |
| (7) | 国家标准规定，图样中汉字应写成 **长仿宋** 体，汉字字宽约为字高 h 的 **0.7** 倍。 |
| (8) | 字体的号数，即字体的 **高度**。"4"号是国家标准规定的字高吗？ **不是**。 |
| (9) | 国家标准规定，可见轮廓线用 **粗实线** 表示；不可见轮廓线用 **细虚线** 表示。 |
| (10) | 在机械图样中，粗线和细线的线宽比例为 **2∶1**。 |
| (11) | 在机械图样中一般采用 **箭头** 作为尺寸线的终端。 |
| (12) | 机械图样中的角度尺寸一律 **水平** 方向注写。 |

### 1-1-4 选择题

## 1-2 填空选择题

### 1-2-1 填空题

| 题号 | 答案 |
|:---:|------|
| (1) | 尺寸标注由 **尺寸界线**、**尺寸线**、**尺寸数字** 和 **尺寸终端** 四要素组成。 |
| (2) | 尺寸界线一般应与尺寸线 **垂直**，必要时允许 **倾斜**。 |
| (3) | 尺寸线不能用其它图线代替，一般也不得与其它图线 **重合** 或画在其 **延长线** 上。 |
| (4) | 同一图样中，尺寸界线之间应 **平行**。 |
| (5) | 标注线性尺寸时，尺寸线必须与所标注的线段 **平行**。 |
| (6) | 线性尺寸的数字一般应注写在尺寸线的 **上方**，也允许注写在尺寸线的 **中断处**。 |
| (7) | 线性尺寸数字的方向，水平数字应 **头朝上**，竖直数字应 **头朝左**。 |
| (8) | 尺寸数字不可被任何图线所通过，否则必须将该图线 **断开**。 |
| (9) | 标注角度的尺寸数字，一律写成 **水平** 方向。 |
| (10) | 标注直径尺寸时，应在尺寸数字前加注符号 **φ**；标注半径尺寸时，加注符号 **R**。 |
| (11) | 标注球面的直径或半径尺寸时，应在符号 φ 或 R 前再加注符号 **S**。 |
| (12) | 标注弧长尺寸时，应在尺寸数字上方加注符号 **⌒**。 |

### 1-2-2 选择题

| 题号 | 答案 |
|:---:|------|
| (13) | 制图国家标准规定，图纸的标题栏必须配置在图框的 **右下角** 位置。 |
| (14) | 制图国家标准规定，图纸中字体的 **字高** 一般代表字体的号数，如 5 号字即指该字体的该值为 5mm。 |
| (15) | 制图国家标准规定，图纸中斜体字字头向右倾斜，与 **水平基准** 成 75° 角。 |
| (16) | 制图国家标准规定，图样中书写的字体必须做到：字体工整、笔画清楚、间隔均匀、排列整齐。汉字应书写 **长仿宋体**，并应采用国家正式公布的简化字。 |
| (17) | 制图国家标准规定，图纸上的各种线型，一般分粗、细两种，粗线的宽度应按图的大小和复杂程度在 0.5～2mm 之间选择，细线的宽度应为粗线宽度的 **1/3**。 |
| (18) | 制图国家标准规定，图样中 **尺寸线** 不能与其它图线重合或画在其延长线上。 |

### 1-2-3 选择题

| 题号 | 答案 |
|:---:|------|
| (19) | 制图国家标准规定，图样中尺寸数字不可被任何图线所通过，当不可避免时必须把 **图线** 断开。 |
| (20) | 制图国家标准规定，标注圆的 **直径** 尺寸时，尺寸线一般应通过该圆圆心。 |
| (21) | 制图国家标准规定，标注 **球面半径** 尺寸时，应在尺寸数字前加注符号 SR。 |
| (22) | 制图国家标准规定，机件的每一尺寸，一般只标注 **一次**。 |
| (23) | 制图国家标准规定，**线性** 尺寸数字不可被任何图线通过，否则必须将该图线断开。 |
| (24) | 制图国家标准规定，机件的真实大小应以图样上 **所注尺寸** 为依据，与图形的大小及绘图的准确度无关。 |

### 1-2-4 选择题

| 题号 | 答案 |
|:---:|------|
| (25) | 制图国家标准规定，机械图样中所标注的尺寸，为该图样所示机件的 **最后完工尺寸**，否则应另加说明。 |
| (26) | 制图国家标准规定，尺寸线用 **细实线** 绘制。 |
| (27) | 制图国家标准规定，尺寸界线用细实线绘制，并应由图形的 **轮廓线、轴线或对称中心线** 处引出。 |
| (28) | 制图国家标准规定，线性尺寸的尺寸数字一般应注写在尺寸线的 **上方**，也允许注写在尺寸线的中断处。 |
| (29) | 制图国家标准规定，标注 **角度** 尺寸时，尺寸线应画成圆弧，该圆弧的圆心是该角的顶点。 |
| (30) | 制图国家标准规定，标注 **弧长** 尺寸时，应在尺寸数字上方加注符号 ⌒。 |
