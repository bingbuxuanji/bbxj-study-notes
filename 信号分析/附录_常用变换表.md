# 附录：常用变换表 —— 傅里叶 · 拉普拉斯 · Z 变换

> **速查手册**：涵盖连续/离散信号分析中最常用的三种变换公式。配合 [信号分析入门教程](信号分析入门教程.md) 使用。

---

## 一、傅里叶变换（Fourier Transform, FT）

### 1.1 定义

| | 正变换 | 逆变换 |
|------|--------|--------|
| **连续 FT** | $\displaystyle X(\omega) = \int_{-\infty}^{+\infty} x(t)\, e^{-j\omega t} dt$ | $\displaystyle x(t) = \frac{1}{2\pi} \int_{-\infty}^{+\infty} X(\omega)\, e^{j\omega t} d\omega$ |
| **离散 FT (DTFT)** | $\displaystyle X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x[n]\, e^{-j\omega n}$ | $\displaystyle x[n] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(e^{j\omega})\, e^{j\omega n} d\omega$ |
| **DFT (N点)** | $\displaystyle X[k] = \sum_{n=0}^{N-1} x[n]\, e^{-j\frac{2\pi}{N}kn}$ | $\displaystyle x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k]\, e^{j\frac{2\pi}{N}kn}$ |

### 1.2 常用傅里叶变换对

| 序号 | $x(t)$ / $x[n]$ | $X(\omega)$ / $X(e^{j\omega})$ | 条件/说明 |
|:---:|------|------|------|
| 1 | $\delta(t)$ | $1$ | 白频谱——所有频率分量强度相等 |
| 2 | $1$ | $2\pi\delta(\omega)$ | 直流信号，只在零频有能量 |
| 3 | $e^{j\omega_0 t}$ | $2\pi\delta(\omega - \omega_0)$ | 单频复指数 → 频谱是一根谱线 |
| 4 | $\cos(\omega_0 t)$ | $\pi[\delta(\omega - \omega_0) + \delta(\omega + \omega_0)]$ | 实偶函数 → 频谱实偶 |
| 5 | $\sin(\omega_0 t)$ | $j\pi[\delta(\omega + \omega_0) - \delta(\omega - \omega_0)]$ | 实奇函数 → 频谱纯虚奇 |
| 6 | $e^{-at}u(t)$ | $\displaystyle \frac{1}{a + j\omega}$ | $a > 0$，单边指数衰减 |
| 7 | $g_\tau(t)$（门函数） | $\displaystyle \tau \cdot \text{Sa}\!\left(\frac{\omega\tau}{2}\right)$ | 宽 $\tau$ 的矩形脉冲 |
| 8 | $u(t)$ | $\displaystyle \pi\delta(\omega) + \frac{1}{j\omega}$ | ⚠️ 最容易漏掉 $\pi\delta(\omega)$！|
| 9 | $\text{sgn}(t)$ | $\displaystyle \frac{2}{j\omega}$ | 符号函数 |
| 10 | $e^{-a|t|}$ | $\displaystyle \frac{2a}{a^2 + \omega^2}$ | $a>0$，双边指数 |
| 11 | $\delta[n]$ | $1$ | 离散单位冲激 |
| 12 | $a^n u[n]$ | $\displaystyle \frac{1}{1 - ae^{-j\omega}}$ | $\vert a\vert < 1$，离散单边指数 |
| 13 | $u[n]$ | $\displaystyle \frac{1}{1-e^{-j\omega}} + \pi\sum_{k=-\infty}^{\infty}\delta(\omega-2\pi k)$ | 离散阶跃 |

### 1.3 傅里叶变换性质

| 性质 | 时域 | 频域 |
|------|------|------|
| **线性** | $a x_1(t) + b x_2(t)$ | $a X_1(\omega) + b X_2(\omega)$ |
| **时移** | $x(t - t_0)$ | $e^{-j\omega t_0} X(\omega)$ |
| **频移** | $x(t) e^{j\omega_0 t}$ | $X(\omega - \omega_0)$ |
| **调制** | $x(t) \cos(\omega_0 t)$ | $\frac{1}{2}[X(\omega+\omega_0) + X(\omega-\omega_0)]$ |
| **尺度** | $x(at)$ | $\frac{1}{|a|} X\!\left(\frac{\omega}{a}\right)$ |
| **时域卷积** | $x_1(t) * x_2(t)$ | $X_1(\omega) \cdot X_2(\omega)$ |
| **频域卷积** | $x_1(t) \cdot x_2(t)$ | $\frac{1}{2\pi}[X_1(\omega) * X_2(\omega)]$ |
| **时域微分** | $\frac{d}{dt}x(t)$ | $j\omega X(\omega)$ |
| **时域积分** | $\int_{-\infty}^t x(\tau)d\tau$ | $\frac{X(\omega)}{j\omega} + \pi X(0)\delta(\omega)$ |
| **共轭对称** | $x^*(t)$ | $X^*(-\omega)$ |
| **对偶** | $X(t)$ | $2\pi x(-\omega)$ |
| **帕塞瓦尔** | $\int |x(t)|^2 dt$ | $\frac{1}{2\pi}\int |X(\omega)|^2 d\omega$ |

### 1.4 傅里叶级数（FS）

周期为 $T$ 的周期信号 $f(t)$：

$$\boxed{f(t) = \sum_{n=-\infty}^{\infty} F_n e^{jn\omega_0 t}} \qquad \boxed{F_n = \frac{1}{T} \int_{-T/2}^{T/2} f(t) e^{-jn\omega_0 t} dt}$$

其中 $\omega_0 = 2\pi/T$。周期矩形脉冲的系数：

$$F_n = \frac{A\tau}{T} \cdot \text{Sa}\!\left(\frac{n\omega_0\tau}{2}\right)$$

**Sa 函数**：$\text{Sa}(x) = \dfrac{\sin x}{x}$（第一过零点 $x=\pi$）

## 二、拉普拉斯变换（Laplace Transform, LT）

### 2.1 定义

$$\boxed{F(s) = \int_{0^-}^{+\infty} f(t)\, e^{-st} dt}$$

其中复变量 $\boxed{s = \sigma + j\omega}$ ：
- $\sigma$ = 衰减因子（实部）
- $\omega$ = 角频率（虚部）
- $\sigma = 0$ 时 → LT 退化为 FT（$s = j\omega$）

> ⚠️ 积分下限取 $0^-$（不是 $0^+$），确保捕获 $t=0$ 处的 $\delta(t)$。

### 2.2 常用拉普拉斯变换对

| 序号 | $f(t)$（$t \ge 0^-$） | $F(s)$ | ROC |
|:---:|------|------|------|
| 1 | $\delta(t)$ | $1$ | 全 $s$ 平面 |
| 2 | $u(t)$ | $\dfrac{1}{s}$ | $\sigma > 0$ |
| 3 | $t$ | $\dfrac{1}{s^2}$ | $\sigma > 0$ |
| 4 | $\dfrac{t^{n-1}}{(n-1)!}$ | $\dfrac{1}{s^n}$ | $\sigma > 0$ |
| 5 | $e^{at}$ | $\dfrac{1}{s-a}$ | $\sigma > a$ |
| 6 | $t e^{at}$ | $\dfrac{1}{(s-a)^2}$ | $\sigma > a$ |
| 7 | $\dfrac{t^{n-1}}{(n-1)!}e^{at}$ | $\dfrac{1}{(s-a)^n}$ | $\sigma > a$ |
| 8 | $\cos(\omega_0 t)$ | $\dfrac{s}{s^2 + \omega_0^2}$ | $\sigma > 0$ |
| 9 | $\sin(\omega_0 t)$ | $\dfrac{\omega_0}{s^2 + \omega_0^2}$ | $\sigma > 0$ |
| 10 | $e^{-at}\cos(\omega_0 t)$ | $\dfrac{s+a}{(s+a)^2 + \omega_0^2}$ | $\sigma > -a$ |
| 11 | $e^{-at}\sin(\omega_0 t)$ | $\dfrac{\omega_0}{(s+a)^2 + \omega_0^2}$ | $\sigma > -a$ |

### 2.3 拉普拉斯变换性质

| 性质 | 时域 | $s$ 域 |
|------|------|------|
| **线性** | $a f_1(t) + b f_2(t)$ | $a F_1(s) + b F_2(s)$ |
| **时移** | $f(t - t_0)u(t - t_0)$ | $e^{-s t_0} F(s)$ |
| **频移** | $e^{s_0 t} f(t)$ | $F(s - s_0)$ |
| **尺度** | $f(at)$ | $\frac{1}{a}F\!\left(\frac{s}{a}\right)$，$a>0$ |
| **时域微分** | $\dfrac{df}{dt}$ | $sF(s) - f(0^-)$ |
| **时域二阶微分** | $\dfrac{d^2f}{dt^2}$ | $s^2F(s) - sf(0^-) - f'(0^-)$ |
| **时域积分** | $\int_{0^-}^t f(\tau)d\tau$ | $\dfrac{F(s)}{s}$ |
| **时域卷积** | $f_1(t) * f_2(t)$ | $F_1(s) \cdot F_2(s)$ |
| **初值定理** | $f(0^+)$ | $\lim_{s\to\infty} sF(s)$ |
| **终值定理** | $f(\infty)$ | $\lim_{s\to 0} sF(s)$（极点全在左半平面时） |

### 2.4 收敛域（ROC）——拉普拉斯变换的灵魂

> 没有 ROC 的 $F(s)$ 是不完整的——同一个 $F(s)$ 配上不同 ROC，对应完全不同的 $f(t)$。

| 信号类型 | ROC 形状 | 规则 |
|---------|---------|------|
| 因果信号（右边信号） | $\sigma > \max\{\text{极点实部}\}$ | 右半平面 |
| 反因果信号（左边信号） | $\sigma < \min\{\text{极点实部}\}$ | 左半平面 |
| 双边信号 | $\sigma_1 < \sigma < \sigma_2$ | **带状** |
| 有限时长信号 | 全 $s$ 平面 | 积分区间有限，一定收敛 |

> 🔑 **ROC 绝不包含极点**——极点是 ROC 的"围墙"。

### 2.5 ROC 与傅里叶变换的关系

$$\boxed{\text{ROC 包含 } j\omega \text{ 轴} \iff \text{傅里叶变换存在}}$$

- $e^{-t}u(t)$：极点 $-1$，ROC $\sigma > -1$ → 含 $j\omega$ 轴 → FT 存在 ✅
- $e^{2t}u(t)$：极点 $2$，ROC $\sigma > 2$ → 不含 $j\omega$ 轴 → FT 不存在 ❌

### 2.6 稳定性判据（$s$ 域）

$$\boxed{\text{因果系统 BIBO 稳定} \iff \text{所有极点实部 } < 0 \text{（左半平面）}}$$

| 极点位置 | 实部 | 时域 $e^{\sigma t}$ | $t\to\infty$ | 稳定？ |
|---------|:---:|------|:---:|:---:|
| 左半平面 | $\sigma < 0$ | $e^{-\alpha t}$ 衰减 | → 0 | ✅ 稳定 |
| 虚轴上 | $\sigma = 0$ | 常数/等幅振荡 | → 常数 | ❌ 临界不稳定 |
| 右半平面 | $\sigma > 0$ | $e^{\alpha t}$ 增长 | → ∞ | ❌ 不稳定 |

### 2.7 部分分式展开：覆盖法（Heaviside Cover-up）

对于 $F(s) = \dfrac{N(s)}{(s-p_1)(s-p_2)\cdots(s-p_n)}$，展开为：

$$F(s) = \frac{A_1}{s-p_1} + \frac{A_2}{s-p_2} + \cdots + \frac{A_n}{s-p_n}$$

系数 $A_k$ 直接用**覆盖法**求得：

$$\boxed{A_k = \left. (s-p_k)F(s) \right|_{s=p_k}}$$

即：盖住分母里的 $(s-p_k)$，把 $s$ 全换成 $p_k$。

## 三、Z 变换（Z Transform, ZT）

### 3.1 定义

$$\boxed{X(z) = \sum_{n=-\infty}^{\infty} x[n]\, z^{-n}}$$

> ⚠️ 指数是 **$-n$**（负号！），这是 Z 变换最容易写错的地方。

- 单边 ZT：$\displaystyle X(z) = \sum_{n=0}^{\infty} x[n] z^{-n}$

### 3.2 常用 Z 变换对

| 序号 | $x[n]$ | $X(z)$ | ROC |
|:---:|------|------|------|
| 1 | $\delta[n]$ | $1$ | 全 $z$ 平面 |
| 2 | $\delta[n-k]$ | $z^{-k}$ | $z \neq 0$（$k>0$） |
| 3 | $u[n]$ | $\displaystyle \frac{1}{1-z^{-1}} = \frac{z}{z-1}$ | $\vert z\vert > 1$ |
| 4 | $a^n u[n]$ | $\displaystyle \frac{1}{1-az^{-1}} = \frac{z}{z-a}$ | $\vert z\vert > \vert a\vert$ |
| 5 | $n a^n u[n]$ | $\displaystyle \frac{az^{-1}}{(1-az^{-1})^2}$ | $\vert z\vert > \vert a\vert$ |
| 6 | $-a^n u[-n-1]$ | $\displaystyle \frac{1}{1-az^{-1}}$ | $\vert z\vert < \vert a\vert$ |
| 7 | $\cos(\omega_0 n) u[n]$ | $\displaystyle \frac{1 - z^{-1}\cos\omega_0}{1 - 2z^{-1}\cos\omega_0 + z^{-2}}$ | $\vert z\vert > 1$ |
| 8 | $\sin(\omega_0 n) u[n]$ | $\displaystyle \frac{z^{-1}\sin\omega_0}{1 - 2z^{-1}\cos\omega_0 + z^{-2}}$ | $\vert z\vert > 1$ |
| 9 | $R_N(n)$（$N$ 点矩形序列） | $\displaystyle \frac{1 - z^{-N}}{1 - z^{-1}}$ | $\vert z\vert > 0$ |

### 3.3 Z 变换性质

| 性质 | 时域 | $z$ 域 |
|------|------|------|
| **线性** | $a x_1[n] + b x_2[n]$ | $a X_1(z) + b X_2(z)$ |
| **时移** | $x[n-n_0]$ | $z^{-n_0} X(z)$ |
| **指数加权** | $a^n x[n]$ | $X\!\left(\frac{z}{a}\right)$ |
| **时域反转** | $x[-n]$ | $X(z^{-1})$ |
| **时域卷积** | $x_1[n] * x_2[n]$ | $X_1(z) \cdot X_2(z)$ |
| **差分** | $x[n] - x[n-1]$ | $(1-z^{-1})X(z)$ |
| **累加** | $\sum_{k=-\infty}^{n} x[k]$ | $\dfrac{X(z)}{1-z^{-1}}$ |
| **初值定理** | $x[0]$ | $\lim_{z\to\infty} X(z)$（因果序列） |
| **终值定理** | $x[\infty]$ | $\lim_{z\to 1} (1-z^{-1})X(z)$（极点全在单位圆内） |

### 3.4 收敛域（ROC）与因果/稳定判据

| 判据 | 条件 | ROC 特征 |
|------|------|------|
| **因果** | $x[n]=0$ 对 $n<0$ | ROC 是圆外：$\vert z\vert > R$ |
| **反因果** | $x[n]=0$ 对 $n>0$ | ROC 是圆内：$\vert z\vert < R$ |
| **稳定** | $\sum \vert x[n]\vert < \infty$ | ROC 包含单位圆 $\vert z\vert = 1$ |
| **因果且稳定** | 极点全在单位圆内 | ROC: $\vert z\vert > \max\vert\text{极点}\vert$，且 $\max\vert\text{极点}\vert < 1$ |
| **有限长序列** | 只在有限 $n$ 处非零 | $0 < \vert z\vert < \infty$（可能不含 0 或 $\infty$） |

> 🔑 **Z 域稳定性**：对因果系统，**所有极点在单位圆内** $\iff$ BIBO 稳定。

### 3.5 逆 Z 变换标准流程

1. **展开 $\dfrac{X(z)}{z}$**（不是直接展开 $X(z)$）
2. 用覆盖法求部分分式系数
3. **乘回 $z$**，化为 $z^{-1}$ 标准形式
4. 查表逆变换，根据 ROC 确定是因果/反因果序列

### 3.6 ZT 与 DTFT / DFT 的关系

| 关系 | 公式 | 含义 |
|------|------|------|
| ZT → DTFT | $X(e^{j\omega}) = X(z)\big|_{z=e^{j\omega}}$ | DTFT = ZT 在单位圆上的取值 |
| ZT → DFT | $X[k] = X(z)\big|_{z=e^{j2\pi k/N}}$ | DFT = ZT 在单位圆上 $N$ 点等间隔采样 |

## 四、三种变换的关系对照

### 4.1 变换域对照

| | 傅里叶变换 FT | 拉普拉斯变换 LT | Z 变换 ZT |
|------|:---:|:---:|:---:|
| **适用信号** | 连续时间 | 连续时间 | 离散时间 |
| **变换变量** | $\omega$（实频率） | $s = \sigma + j\omega$（复频率） | $z = re^{j\omega}$（复变量） |
| **积分/求和区间** | $(-\infty, +\infty)$ | $0^- \to +\infty$（单边） | $-\infty \to +\infty$（双边） |
| **核心优势** | 直观的频率分析 | 可分析不稳定系统 | 离散系统分析 |
| **存在条件** | 绝对可积 | 合适的 $\sigma$ 使积分收敛 | 合适的 $r$ 使级数收敛 |

### 4.2 核心信号的三域对比

| 信号 | FT | LT | ZT（离散对应） |
|------|:---:|:---:|:---:|
| $\delta(t)$ / $\delta[n]$ | $1$ | $1$ | $1$ |
| $u(t)$ / $u[n]$ | $\pi\delta(\omega)+\frac{1}{j\omega}$ | $\frac{1}{s}$ | $\frac{1}{1-z^{-1}}$ |
| $e^{-at}u(t)$ / $a^n u[n]$ | $\frac{1}{a+j\omega}$ | $\frac{1}{s+a}$ | $\frac{1}{1-az^{-1}}$ |
| $\cos(\omega_0 t)$ / $\cos(\omega_0 n)$ | 冲激对 | $\frac{s}{s^2+\omega_0^2}$ | 有理分式 |
| $\sin(\omega_0 t)$ / $\sin(\omega_0 n)$ | 冲激对 | $\frac{\omega_0}{s^2+\omega_0^2}$ | 有理分式 |

### 4.3 稳定性判据对照

| 域 | 稳定条件 |
|------|------|
| FT（$s=j\omega$） | ROC 必须包含虚轴 |
| LT（$s$ 平面） | 极点全在**左半平面**（$\sigma < 0$） |
| ZT（$z$ 平面） | 极点全在**单位圆内**（$\vert z\vert < 1$） |

### 4.4 时频对偶关系

$$\boxed{\text{时域周期} \;\longleftrightarrow\; \text{频域离散}}$$
$$\boxed{\text{时域非周期} \;\longleftrightarrow\; \text{频域连续}}$$
$$\boxed{\text{时域离散} \;\longleftrightarrow\; \text{频域周期}}$$
$$\boxed{\text{时域连续} \;\longleftrightarrow\; \text{频域非周期}}$$

| 时域 | $\longleftrightarrow$ | 频域 |
|:---:|:---:|:---:|
| 周期 | ↔ | **离散** |
| 非周期 | ↔ | **连续** |
| 离散 | ↔ | **周期** |
| 连续 | ↔ | **非周期** |

### 4.5 变换间的数学桥梁

| 关系 | 公式 |
|------|------|
| LT 退化为 FT | $\mathcal{F}\{f(t)\} = \mathcal{L}\{f(t)\}\big|_{s=j\omega}$（ROC 含 $j\omega$ 轴） |
| ZT 退化为 DTFT | $\text{DTFT}\{x[n]\} = \mathcal{Z}\{x[n]\}\big|_{z=e^{j\omega}}$（ROC 含单位圆） |
| DFT 采样 ZT | $X[k] = X(z)\big|_{z=e^{j2\pi k/N}}$ |
| $s$ 平面 $\to$ $z$ 平面 | $z = e^{sT}$（脉冲响应不变法） |
| $s$ 平面 $\to$ $z$ 平面 | $s = \frac{2}{T}\frac{1-z^{-1}}{1+z^{-1}}$（双线性变换法） |

## 五、常用数学关系

### 5.1 欧拉公式

$$\boxed{e^{j\theta} = \cos\theta + j\sin\theta}$$

$$\cos\theta = \frac{e^{j\theta} + e^{-j\theta}}{2} \qquad \sin\theta = \frac{e^{j\theta} - e^{-j\theta}}{2j}$$

### 5.2 Sa 函数（采样函数）

$$\boxed{\text{Sa}(x) = \frac{\sin x}{x}}$$

性质：
- $\text{Sa}(0) = 1$
- $\text{Sa}(n\pi) = 0$（$n = \pm 1, \pm 2, \dots$）——**第一过零点在 $x = \pi$**
- $\int_{-\infty}^{+\infty} \text{Sa}(x)dx = \pi$

### 5.3 $\delta$ 函数核心性质

| 性质 | 公式 |
|------|------|
| 筛选 | $\int f(t)\delta(t-t_0)dt = f(t_0)$ |
| 尺缩 | $\delta(at) = \frac{1}{\vert a\vert}\delta(t)$ |
| 卷积单位元 | $\delta(t) * f(t) = f(t)$ |
| $u$ 的导数 | $\frac{du(t)}{dt} = \delta(t)$ |
| $\delta'$ 筛选 | $\int f(t)\delta'(t)dt = -f'(0)$ |

### 5.4 几何级数（无穷等比数列求和）

$$\boxed{\sum_{n=0}^{\infty} r^n = \frac{1}{1-r}, \quad |r| < 1}$$

这是 DTFT 和 Z 变换展开的核心工具。

### 5.5 部分分式展开配方

| 分母形式 | 展开形式 |
|------|------|
| $(s-p_1)(s-p_2)$ 单实根 | $\frac{A}{s-p_1} + \frac{B}{s-p_2}$ |
| $(s-p)^2$ 重根 | $\frac{A}{s-p} + \frac{B}{(s-p)^2}$ |
| $(s^2 + as + b)$ 共轭复根 | $\frac{As + B}{s^2 + as + b}$（保留二次因式） |

## 六、十大易错点

| # | 易错点 | 正确结论 |
|:---:|------|------|
| 1 | $\delta$ 是 $u$ 的积分 | ❌ $\delta = du/dt$（$\delta$ 是 $u$ 的**导数**） |
| 2 | $\mathcal{F}\{u(t)\} = 1/j\omega$ | ❌ 漏了 $\pi\delta(\omega)$ |
| 3 | ZT 指数是 $z^k$ | ❌ 是 **$z^{-k}$**（负号！） |
| 4 | 周期 ↔ 连续 | ❌ 周期 ↔ **离散** |
| 5 | 双线性变换是线性映射 | ❌ $\Omega = \frac{2}{T}\tan\frac{\omega}{2}$（**非线性**） |
| 6 | $\mathcal{L}\{\delta(t)\} = 1/s$ | ❌ = **$1$**（$1/s$ 是 $u(t)$ 的 LT） |
| 7 | LT 积分下限 $0^+$ | ❌ 应取 **$0^-$**（捕获 $\delta(t)$） |
| 8 | ROC 可含极点 | ❌ ROC **绝不包含极点** |
| 9 | $|a|>1$ 时域压缩 → 频域也压缩 | ❌ $|a|>1$：时域压缩 → 频域**扩展** |
| 10 | 逆 ZT 直接展开 $X(z)$ | ❌ 应先展开 **$X(z)/z$**，再乘回 $z$ |
